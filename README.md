# BB84-Encryption-Protocol
Quantum Key Distribution using the BB84 protocol,  implemented in Python with Qiskit
Simulation of the BB84 Quantum Key Distribution protocol using IBM's Qiskit framework. Demonstrates secure key exchange between two parties (Alice & Bob) and eavesdropper detection via Quantum Bit Error Rate (QBER) analysis. Built as a mini project for AIM3231 Emerging Tools & Technologies Lab, Manipal University Jaipur.
